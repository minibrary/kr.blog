명령어를 반복 실행하기 위해 편리한 방법 입니다.

```shell
for i in `seq COUNT`; do COMMAND; done
```

예를 들어, 테스트를 위해 메일을 10번 보내기 위해 아래의 명령어를 실행해 보겠습니다.

```shell
for i in `seq 10`; do sendmail -t < mail.txt; done
```
